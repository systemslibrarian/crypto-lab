# crypto-lab

> Real cryptography. Real primitives. All of it runs in your browser.

A single-page directory of browser-based cryptography demonstrations — from deniable encryption to post-quantum algorithms. No frameworks, no backends, no accounts.

**Live:** https://systemslibrarian.github.io/crypto-lab/

---
